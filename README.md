## INSTALLATION

1. Clone the repo
   ```sh
   git clone https://github.com/PiotrSierant/OMNI_translateAPP.git
   ```
2. Install NPM packages and start project

   ```sh
   npm install
   ```

3. Start project
   ```sh
   npm start
   ```
